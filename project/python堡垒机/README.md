1. 什么是堡垒机？
就是使用一台服务器通过各种方式对远程进行控制和规范。其实因为批量操作而出现的。
2. 堡垒机的实现
通过堡垒机远程登陆到服务器，其应该具有远程登陆（ssh）的能力，如果一台服务器接收堡垒机的管理，就要求该服务器的防火墙对其他的除堡垒机之外的任何ssh进行拒绝。
